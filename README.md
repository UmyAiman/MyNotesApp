# MyNotesApp 

MyNotesApp is a React Native mobile application developed for **Mobile Application Development**.  
The app allows users to easily **add, delete, and search notes** using a connected Firebase database.

 Features

- Add new notes
- Delete existing notes
- Search notes by keywords
- Firebase database integration
- Simple and user-friendly interface
- Android APK support
